# KiraAI_xml_tag_fixer_plugin/掉消息标签修复插件1.06

修复AI未正确包裹标签导致用户无法看到消息的问题，自动补全 AI 输出中缺失的 &lt;msg> 和 &lt;text> 标签

0.功能介绍：

该插件通过自动后处理的方式，辅助KiraAI官方的xml修复，修正一些AI模型（例如deepseek）经常不正确带<text><msg>标签导致用户无法看到消息的问题。

简单说，装了这个能大大减少d指导不带标签直接发消息你看不到的神秘情况！

同时，修复一些模型因渠道限制、降智等缘故兼容xml格式失败而产生错误的双尖括号问题（例如&lt;&lt;msg>）和at效果不真实问题。

1.安装和使用方法：

解压成文件夹放入data\plugins\即可。首次安装后，推荐重开一次会话，否则会不正确生效（例如ai说话中间带空行而不是拆分消息）

2.已知的bug：

（1）私聊时，如果启用了输入状态相关插件，当AI主动戳一戳你时，其输入状态显示可能无法停止，需要你再跟ta对话一次（再调用一次llm）来结束。

（2）可能有一些使用场合，一些无需包裹<text>标签的内容也被包裹，我个人尽量进行了忽略写法，但可能依然存在一些没考虑到的情况，欢迎反馈。
